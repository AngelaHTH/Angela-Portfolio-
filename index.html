Project 3: Business Analysis Project

Data analysis of housing trends and birth rates in Singapore

Tableau Data Visualization Statistical Analysis Dashboard Design Python

Overview

Led a team of four in a data-driven investigation into Singapore's declining birth rate. The project aimed to uncover and visualise the relationships between housing trends and family formation decisions to provide actionable insights.

Key Features & Analysis Highlights

  • Developed an interactive Tableau dashboard analysing the impact on birth rates across three dimensions

My Role

Project 4: Understanding Data Project

Environmental sustainability data analysis for Singapore

Microsoft Excel Data Cleaning Pivot Tables Data Analysis Statistical Analysis

Overview

Executed a comprehensive, solo data analysis to investigate Singapore's environmental sustainability indicators from 2007 to 2022. The project involved synthesizing complex, multi-year datasets into clear insights on waste management, renewable energy adoption, greenhouse gas emissions, and climate-related factors.

Key Features & Analysis Highlights

  • Conducted an end-to-end analysis across multiple environmental dimensions using cleaned and pivoted data
  • Waste Management: Analysed trends in total waste generated, recycled, and disposed. Drilled down into recycling rates by material type (e.g., wood, paper, plastic) to identify areas of strength and opportunity
  • Energy Transition: Tracked the exponential growth of solar panel installation as a key metric for Singapore's shift towards renewable energy
  • Emissions Profile: Investigated the composition of greenhouse gas emissions over time, noting the dominant share of CO₂ and the changing contributions of other gases like HFCs and PFCs
  • Climate & Environmental Factors: Correlated data on sea level rise, temperature extremes, rainfall patterns, and the stability of protected land areas to build a holistic view of environmental pressures
  • Ancillary Data Integration: Incorporated datasets on public transport usage to explore potential correlations between mobility trends and environmental outcomes

My Role

I was responsible for the entire data pipeline: data auditing, cleaning, transformation (using pivot tables), exploratory analysis, synthesis of findings, and the creation of structured summary tables for clear presentation.

Tools & Technologies Used

Microsoft Excel (Data Cleaning, Pivot Tables, Data Analysis), Statistical Analysis, Data Synthesis

Project 5: Velvet Oasis Hotel

High-fidelity UI/UX design for luxury hotel website

Figma UI Design UX Research Prototyping Design System

Overview

Designed and developed a high-fidelity, interactive web prototype for "Velvet Oasis," a luxury hotel. The project focused on creating a seamless user journey from initial discovery to finalized booking, ensuring a sophisticated aesthetic that aligns with a premium hospitality brand.

Key Features & Design Highlights

  • Comprehensive User Interface: Developed intuitive navigation and responsive design elements for optimal user experience across all devices.
  • Integrated Booking System: Built a multi-step booking funnel for both room stays and event venues. Included date pickers, room selection carousels, and dynamic "Total Price" calculations.
  • User Account Management: Designed secure entry points for Sign In, Sign Up, and "Forgot Password" flows, incorporating real-time form validation with password strength indicators and error messaging.
  • Interactive FAQ & Support: Implemented a categorized FAQ section (Payment, Service, General) with accordion-style interaction and a simulated chatbot interface to enhance customer self-service.
  • Visual Property Showcases: Created detailed "Hotel Room View" pages featuring high-quality image galleries, categorized amenities (Complimentary Services), and customer testimonials to build brand trust.
  • Payment Gateway Integration: Prototyped a comprehensive checkout experience supporting multiple payment methods, including PayNow (QR code simulation) and Credit/Debit card validation flows.

My Role

I was responsible for the end-to-end creative process: user flow mapping, wireframing, component-based design system creation, interactive prototyping, and usability testing for form validations and navigation logic.

Tools & Technologies Used

Figma (Auto Layout, Components & Variants, Advanced Prototyping), UI Design, UX Research, Interaction Design, Design Systems, Usability Testing.

Outcome

Delivered a fully functional interactive prototype that effectively simulates a live luxury hotel website. The final design provides a cohesive brand identity and a friction-less booking experience, validated through complex state-management within Figma.
